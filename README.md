# HR_SHEMA-MYSQL

📌 HR Schema 

* The HR schema is a sample database in Oracle Database used for learning and practicing SQL.

It represents a Human Resources system with tables like:

* Employees (employee details)
* Departments (department info)
* Jobs (job roles)
* Locations, Countries, Regions (geographical data)
* Job_History (employee career changes)

👉 All tables are connected using keys, which helps practice:

* JOINs
* Filtering (WHERE)
* Grouping (GROUP BY)
* Aggregate functions (COUNT, MAX, etc.)
🎯 Purpose
* Learn SQL easily
* Practice real-world data queries
* Prepare for Data Analyst interviews
